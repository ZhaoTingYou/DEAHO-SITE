> 각 파일은 웹 개발에서 **파일명 그대로** 사용하므로 **영문 파일명은 절대 변경하지 말아 주세요.** (설명만 한국어)

## 0. 전체 공통 규칙 (모든 컷에 적용)

**브랜드 톤 (가장 중요)**
- 전체 톤은 **밝고 환한(high-key) 럭셔리**. 따뜻한 아이보리·순백·밝은 베이지 배경, 부드러운 자연광, 부드럽고 긴 그림자.
- **검정/어두운 배경 절대 사용 금지.** 무드 어둡게 X, 음영 과하게 X.
- 금·로즈골드·은·보석이 따뜻한 빛 속에서 은은하게 반짝이도록.
- 포인트 색(아주 조금만): 네이비(#101D30), 와인레드(#7A2230). 큰 면적으로 쓰지 말 것.
- 레퍼런스 무드: Cartier / Patek Philippe / Omega 의 **밝은 제품·매장 컷**.

**촬영/기술 사양**
- 색감: 따뜻한 화이트 밸런스(차가운 푸른 흰색 X). 배경은 순백보다 **약간 따뜻한 아이보리** 권장.
- 해상도: **가로 기준 최소 2400px 이상**(히어로/풀화면 컷은 **3840px(4K) 이상** 권장).
- 포맷: 원본 **JPG(고화질) 또는 PNG**, 가능하면 **RAW도 함께**. 누끼(배경 분리)가 필요한 컷은 **투명 PNG**도 부탁.
- 색공간: sRGB.
- 비율(aspect ratio)은 각 컷마다 명시 — **그 비율에 맞게 여유 있게** 촬영(후보정 크롭 여유 포함).
- **글자/각인 텍스트는 실제로 새기지 않아도 됨** — 표면은 비워두고, 글자는 후반 작업에서 합성. (AI/합성 모두 글자 정확도가 떨어지므로)
- 로고(구단/협회 등)는 **임의로 넣지 말 것** — 실제 로고는 추후 별도 처리.
- 사람: 제품 컷에는 **얼굴 노출 금지**(손·손목까지만). 라이프스타일 컷도 얼굴은 안 나오게.

**비율별 픽셀 권장치(가로×세로, 최소)**
- 21:9 → 3840×1646 (히어로/풀폭)
- 16:9 → 2560×1440
- 4:3 → 2400×1800
- 3:2 → 2400×1600
- 4:5 (세로) → 1600×2000
- 3:4 (세로) → 1500×2000
- 1:1 (정사각) → 2000×2000

**파일 위치/네이밍**
- 이미지: `파일명.png` (이 문서의 영문명 그대로)
- 영상: `home_hero.mp4` + `home_hero.webm`
- 같은 묶음(예: home_ring_01~05, collection_ring_01~12)은 **조명·배경·각도 톤을 통일**해서 한 세트처럼 보이게.
